Complete website redesign for Rodriguez Laundry, a Nantucket-based linen and laundry pickup/delivery service. Goal: Create a high-conversion, modern website that drives scheduling and establishes trust with both residential and commercial clients.

## PART 2: NEW DESIGN SPECIFICATIONS

### Design Philosophy
Modern, clean, trust-building design optimized for conversions. Primary goal: Get visitors to schedule a pickup within 30 seconds of landing.

### Color Palette

| Role | Color | Hex | Usage |
|------|-------|-----|-------|
| **Primary** | Deep Blue | #1E3A8A | CTAs, headers, primary buttons |
| **Secondary** | Royal Blue | #2563EB | Links, secondary buttons, accents |
| **Tertiary** | Medium Blue | #3B82F6 | Icons, highlights, hover states |
| **Light** | Sky Blue | #60A5FA | Badges, tags, subtle accents |
| **Pale** | Ice Blue | #DBEAFE | Section backgrounds, cards |
| **Neutral Dark** | Slate Blue | #1E293B | Body text, headings |
| **Clean White** | Pure White | #FFFFFF | Primary backgrounds |

*Rationale: Monochromatic blue palette conveys trust, cleanliness, and professionalism. The gradient from deep to pale blue creates visual hierarchy while maintaining a fresh, clean aesthetic appropriate for a laundry service.*

### Mobile-First Considerations

1. **Sticky Header:** Phone number + "Schedule" button always visible
2. **Thumb-Friendly CTAs:** Minimum 48px tap targets
3. **Click-to-Call:** Phone number triggers dialer
4. **Simplified Navigation:** Hamburger menu with max 5 items
5. **Single Column:** All content stacks vertically
6. **Reduced Hero Text:** Shorter headlines on mobile
7. **Swipeable Testimonials:** Card carousel for social proof

### Conversion Optimization Elements

| Element | Implementation |
|---------|----------------|
| **Primary CTA** | "Schedule Free Pickup" - appears 4x on page |
| **Secondary CTA** | Phone number with click-to-call |
| **Urgency** | "Next-day delivery available" badge |
| **Trust Signals** | Chamber member, years in business, review count |
| **Exit Intent** | Popup offering 10% off first order |
| **Live Chat** | Optional - or SMS text option |
